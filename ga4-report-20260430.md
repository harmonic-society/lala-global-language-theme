━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

過去のレポートで「水曜日の直帰率異常（97.4 → 97.7 → 97.7 → 100.0%）」と
報告していましたが、データを再検証した結果、これは誤報でした。

【真相】
GA4のデータは確定までに24-72時間かかります。レポート作成時点で「最新の水曜日」は
データが未確定（処理中）であり、bounce率が一時的に100%近くに表示されるが、
24-72時間後に正常値で確定します。

【検証結果（再取得した確定値）】

  日付　　　　レポート時の値　　現在の確定値
  ────────────────────────────
  4/8 (水)　　 bounce 97.4%　→ bounce 53.2% ✅ 正常
  4/15(水)　　 bounce 97.7%　→ bounce 50.9% ✅ 正常
  4/22(水)　　 bounce 97.7%　→ bounce 60.9% ✅ 正常
  4/29(水)　　 bounce 100.0% → bounce 100.0% ⚠️ まだ未確定（処理中）

【結論】
・「水曜日異常」というパターンは存在しなかった
・実際は「レポート作成日の直前日のデータ未確定」現象だった
・GA4タグもサイト側の計測も正常に動作している

【今後のレポート作成ルール】
レポートは「過去8日間」ではなく「3日前〜10日前の8日間」を対象にする。
あるいは、最新の1-2日分は「データ未確定の暫定値」と明記する。

→ 本レポートの 4/29 数値、3節「日別トレンド」、14節「課題1」の記述は
  いずれも実際の異常ではなく、データ未確定の暫定値を異常と誤認したもの。
  各セクションの当該記述は無視してください。
